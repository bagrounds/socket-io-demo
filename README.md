# socket-io-demo

## Get the project

```sh
# get the project
git clone git@gitlab.com:bagrounds/socket-io-demo

# change directories
cd socket-io-demo
```

## Install npm dependencies

```sh
npm install
```

## Start the server

```sh
npm start

# direct your browser to localhost:8080
```

or

```sh
PORT=1234 npm start

# direct your browser to localhost:1234
```
