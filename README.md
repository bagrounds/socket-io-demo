# socket-io-demo

## Running the demo

```sh
# get the project
git clone git@gitlab.com:bagrounds/socket-io-demo

# change directories
cd socket-io-demo

# install npm dependencies
npm install

# start the server
npm start

# direct your browser to localhost:8080
```

or

```sh
# get the project
git clone git@gitlab.com:bagrounds/socket-io-demo

# change directories
cd socket-io-demo

# install npm dependencies
npm install

# start the server
PORT=1234 npm start

# direct your browser to localhost:1234
```
